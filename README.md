# EMSHOP-官方系统源码发布

一个基于 PHP 的轻量级高效内容管理系统（CMS），具有简洁、安全、轻量化的特点。


## 主要功能

### 1. 用户管理
- 多用户角色支持：
  - 超级管理员：拥有系统所有管理权限
  - 管理员：拥有站点全部管理权限（可管理用户、系统设置等）
  - 内容编辑：负责全站文章、资源、评论的管理
  - 注册用户：可发文投稿、管理自己的文章/图文资源
  - 游客：仅浏览网站内容
- 用户注册与登录：
  - 支持后台注册
  - 可配置图形验证码
- 用户权限管理：
  - 限制用户上传内容资源的权限


### 2. 内容管理
- 文章管理：
  - 支持文章分类、标签
  - 支持草稿保存、文章置顶
- 评论管理：暂无
- 媒体资源管理：
  - 支持图片、音频、视频等多媒体文件
  - 资源分类管理、上传与编辑


### 3. 商城功能
- 商品管理：
  - 商品列表、分类、规格
  - 库存管理
- 订单管理：
  - 订单状态跟踪


### 4. 系统设置
- 站点信息配置
- SEO 优化设置
- 服务配置
- 外观管理：
  - 模板管理
  - 导航菜单设置
- 插件扩展：
  - 安装与管理
  - 支持功能扩展


## 技术特点
- 基于 PHP 开发，遵循 MVC 架构
- 支持 MySQL 数据库
- 模板引擎渲染，支持主题定制
- RESTful API 接口支持
- 响应式界面设计，支持移动端访问
- 支持插件扩展，可自定义功能


## 环境要求
- PHP >= 7.4
- MySQL >= 5.6
- 需启用的 PHP 扩展：PDO、GD、Curl


## 安装使用
1. 下载源码，上传到网站根目录
2. 配置 Nginx 伪静态规则（ThinkPHP 规则）
3. 访问网站域名，按照安装向导完成配置
4. 登录后台，开始使用
